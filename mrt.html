MusicReleaseTracker

MusicReleaseTracker is an open-source tool that helps you stay up-to-date on new music. With MRT, there's no need for streaming service accounts or relying on machine learning recommendation. MRT's purpose is to save you time and sanity. Developed with the backdrop of absolute bangers and ❤️

Combination of multiple sources ensures reliable information and independency on any one. Available for windows and linux.

If you’re looking for a desktop program to help keep you up to date with the latest releases from your loved artists, then MusicReleaseTracker should make for a good option to consider. Though rather simple, I found MusicReleaseTracker to be rather effective at what it does.
